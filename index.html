Projects

  • PostShed
    Webapp for drafting posts to reddit visually and posting them on schedule.

  • Frame
    CLI for framing images with whitespace (or other colour) to a desired ratio. Fits the /r/fujifilm aesthetic.

  • Caption
    CLI to apply to an image a user-added custom caption and a bit of metadata (pulled automatically from the image file) in customizable colours and arrangements.

  • Epub Converter
    REST API to convert .md files to epub using pandoc. Supports Markdown and text files as sources and Lua filters for templates.

  • TFSA Balancer
    A CLI or GUI python app to balance a portfolio to a desired percentage, with options for full-balance (to include selling) or regular, incremental investments only.

  • EES
    Utility for EPUB checking, finding production issues which aren't within the purview of epubchecker. Includes production and qa tools for encrypting fonts and comparing epub revisions using diffs for each file within.

  • pdf2fxl
    Python script to convert pdf image-only files into fixed-layout epubs.
